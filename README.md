**Stack:** FlutterFlow, Supabase, PostgreSQL, SQL, geolocation, JSON data, custom actions, payment flow.

Built user roles, service search by distance, master availability logic, location updates, backend structure, service list logic, and payment flow.

---

### Brandi Flo  
E-commerce mobile app for branded products.

**Stack:** FlutterFlow, Firebase, Firestore, Firebase Storage, REST APIs, push notifications, deep links, payment integration, App Store / Google Play.

Built product catalog, filters, wishlist, order history, delivery statuses, user profile, admin logic, payment flow, and production release flow.

### Olimp Tickets  
Bus ticket booking mobile app.

**Stack:** FlutterFlow, Firebase, Firestore, Cloud Functions, Google Maps, push notifications, deep links, payment integration.

Built route search, stop selection, ticket creation logic, price calculation, payment flow, deep link return, admin logic, and production support.
